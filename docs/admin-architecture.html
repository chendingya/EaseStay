运行时数据流视图

A. 登录与鉴权链路
登录页提交→ POST /api/auth/login→ token + role 落盘→ 路由守卫放行→ 菜单按角色渲染
B. 列表查询链路
筛选器变更→ useRemoteTableQuery→ api.get()→ /api/admin/hotels?page=...→ Table 渲染
C. i18n 路由链路
路由切换→ getRouteNamespaces()→ loadNamespaces()→ React.lazy 页面加载→ 多语言页面呈现
D. 审核动作链路
审核页提交操作→ PUT/PATCH 审核接口→ 后端更新状态→ 通知中心计数刷新→ 列表结果回流
对应接口域：`/api/merchant/*`、`/api/admin/*`、`/api/notifications`。
这张图强调“角色 + 路由 + 请求 + 国际化”四条主线如何协同运行。
