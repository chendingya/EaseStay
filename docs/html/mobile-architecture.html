运行时数据流视图

A. 查询链路
首页查询触发→ 列表页参数组装→ api.get()→ /api/hotels→ 酒店卡片渲染
B. 下单支付链路
详情页点击预订→ POST /api/hotels/:id/orders→ 跳转 order-pay→ POST /api/user/orders/:id/pay→ 订单状态刷新
C. 用户态链路
登录/注册→ Token 持久化→ 请求拦截注入→ UserContext 同步→ 页面权限更新
D. 图片链路
原始图片 URL→ resolveImageUrl()→ /api/image 代理→ 参数化返回→ 前端展示
对应后端接口族：`/api/auth`、`/api/hotels`、`/api/user`、`/api/image`。
这一视图重点表达“UI 事件如何变成真实请求并回流到渲染”。
