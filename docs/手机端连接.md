# 移动端 H5 本地联调（手机访问）

## 1. 启动 H5 开发服务

在 `mobile` 目录下执行：

```bash
npm install
npm run dev:h5
```

当前开发服务已在 `config/dev.js` 里配置为：

- Host：0.0.0.0
- Port：10086

这表示同一局域网内的手机可以直接访问。

## 2. 获取电脑局域网 IP

在电脑上查看本机局域网 IP（例如 `192.168.50.202`）。

Windows 可在终端执行：

```bash
ipconfig
```

Mac 可在终端执行：

```bash
ifconfig
```

## 3. 手机访问

确保手机和电脑在同一 Wi-Fi，然后在手机浏览器访问：

```
http://<电脑局域网IP>:10086
```

示例：

```
http://192.168.50.202:10086
```

如果访问失败，请检查：

- 电脑防火墙是否允许 10086 端口
- 手机是否处于同一局域网
- 是否开启了 VPN/代理（可能导致局域网访问失败）

## 4. 配置后端接口地址（必须能被手机访问）

移动端接口地址来自 `TARO_APP_API_BASE`，当前在 `.env.local` 中配置：

```
TARO_APP_API_BASE=http://192.168.50.202:4100
```

请确保：

- 该 IP 是电脑的局域网 IP
- 后端服务在该 IP 的 4100 端口已启动

修改 `.env.local` 后需要重启 `npm run dev:h5` 才会生效。
